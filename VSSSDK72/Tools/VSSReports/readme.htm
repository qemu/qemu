Instructions for investigating issues

 1) Copy VSSREPORTS.CAB on the target machine

 2) Unpack all files from VSSREPORTS.CAB in a separate directory.

 3) Run GENERATE_VSS_REPORTS.CMD

 4) Follow instructions

 5) Send the generated file to Microsoft

 

 

Warnings

 The tool assumes that no snapshot creation tests will be done on the %systemroot% volume.

 

 

Included tools

 VSSAGENT.EXE   - data gathering tool used by GENERATE_VSS_REPORTS.CMD

 CABARC.EXE     - CAB creation utility used by GENERATE_VSS_REPORTS.CMD

 VSHADOW.EXE    - additional tool.

 VSDIAGVIEW.EXE - additional tool.
