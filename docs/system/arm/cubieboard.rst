Cubietech Cubieboard (``cubieboard``)
=====================================

The ``cubieboard`` model emulates the Cubietech Cubieboard,
which is a Cortex-A8 based single-board computer using
the AllWinner A10 SoC.

Emulated devices:

- Timer
- UART
- RTC
- EMAC
- SDHCI
- USB controller
- SATA controller
- TWI (I2C) controller
